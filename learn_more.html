Welcome to the curriculum of our class. This is where you will see all the things that we teach throughout the duration of this free tutoring. Most sessions will be posted on our youtube channel for people who missed. Please read the curriculum and consider signing up for our course.

Module 1: Introduction to Python

Week 1:

What is Python?

Setting up Python on your computer (Python installation)

Writing your first Python program (Hello, World!)

Variables and data types (integers, floats, strings, and booleans)

Week 2:

Basic input and output Comments and code formatting Introduction to Python's built-in functions and modules Module 2: Control Structures Week 3: Conditional statements (if, elif, else) Boolean expressions and operators Using comparison and logical operators Week 4: Loops (for and while) Iterating through lists and strings Break and continue statements List comprehensions Module 3: Data Structures Week 5: Lists and their methods Tuples and sets Dictionaries and dictionary methods Week 6: Introduction to objects and classes Creating and using custom classes Object-oriented programming (OOP) concepts Module 4: Functions and Modules Week 7: Defining and calling functions Parameters and return values Scope and lifetime of variables Week 8: Importing and using modules Creating and organizing your own modules Understanding namespaces and name resolution Module 5: File Handling Week 9: Reading and writing text files Working with CSV and JSON data Exception handling (try, except, finally) Module 6: Advanced Topics Week 10: List comprehensions and generator expressions Lambda functions and anonymous functions Decorators and decorators for functions Week 11: Object-oriented programming (OOP) concepts in depth Inheritance and polymorphism Design patterns in Python Module 7: Python Libraries and Frameworks Week 12: Introduction to popular Python libraries (e.g., NumPy, Pandas) Data manipulation and analysis Introduction to web frameworks (e.g., Flask, Django) Module 8: Final Projects Weeks 13-16: Work on practical Python projects Apply the knowledge gained throughout the course Receive feedback and guidance from instructors or peers Module 9: Additional Topics (Optional) Week 17+: Depending on your interests and career goals, explore topics such as data visualization, machine learning, web development, or game development using Python. Assessment: Regular quizzes, coding exercises, and assignments throughout the course. A final project to demonstrate practical skills and understanding.
